HTML trivia

HTML tells your browser how to display a webpage (*in today's world we also have javascript for dynamic contents -- think of the Wordle Example)

HTML IS very forgiving on syntax...so you can have typos, extra/missing spaces, etc...and often it will still work just like this


HTML Easter Eggs

Hitting Ctrl+U opens up the page's source code, or ctrl+shift+i to see which element corresponds to which line(s) of HTML

This is also where a lot of "techies" hide secret messages -- including Mozilla and a US Digital Service

Ctrl+U after opening https://www.mozilla.org/en-US/ or https://www.usds.gov/
